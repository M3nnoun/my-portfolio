# 🚲 Bike Sharing Analytics Dashboard

**Duration:** Jan 2023 – Mar 2023  
**Tools Used:** Power BI, Excel, SQL

## 📊 Project Overview

Developed an interactive **Power BI dashboard** to explore and visualize key metrics from a bike-sharing dataset. The dashboard includes:

- Total number of trips
- Average speed of bikes
- Weather-based usage patterns
- Trend analysis over time
- Distribution of bike usage across different weather conditions
- Percentage breakdown of usage during weekends and holidays

This project demonstrates effective data cleaning, transformation, and insightful visualizations that help stakeholders quickly identify and act on trends in bike-sharing behavior.
